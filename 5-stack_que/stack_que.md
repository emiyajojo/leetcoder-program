# 有效括号
遍历候选字符串:
遇到三种左括号, 那么就将相应的右括号加入栈
遇到三种右括号,如果栈里顶端也就是最外面的的元素与现在遍历的元素不相等,返回False
或者,如果栈已经空了并且当前遍历元素是右括号,False

# 逆波兰
用一个暂时变量存储中间运算值
若运算值为NOne,那么当进栈了一个运算符,就pop出两个值
若非空,则pop出1个值,并与中间值运算
接下来是易错关键：将所谓的中间值塞进栈中！所以并不需要额外的中间值！

# 滑动窗口最大值
思路：使用单调队列，从大到小，并且这个队列不需要维持窗口长度的数量的值
遇到小的值就留下，遇到大的值就把队列的尾巴值全去掉
初始化和边界条件的特别处理，看看能不能放进循环内一并完成，比如本函数‘初始化’第一个窗口，还有单调队列的维护判断队列是否为空

# 前k个高频元素
使用小顶堆+字典解决:
1. 字典统计频数
2. 小顶堆构建+维护，这里借助元组的比较方式进行小顶堆的值的相对比较，同时当小顶堆容量不足就弹出
3. 将小顶堆倒序输出到结果数组
   
或者直接用字典统计元素频数,然后用另一个字典反向统计频数:频数为键,相应的元素的列表为值,然后将第二个字典按照键排序,然后再遍历输出前K个高频
